Numerical Values

  • Integers: integer
    border-radius: 15px; z-index: 1; order: 3;
  • Numbers: number
    line-height: 1.5; flex: 0.618;
  • Percentages: percentage
    width: 80%; font-size: 120%;

Color Values

  • Keywords
    lightgreen, gold, olivedrab, etc.
  • Hexadecimal notation
    #69c, #abcdef, #eee, #123456 ...
  • Functional notation
    rgb(200, 150, 100), rgba(200, 150, 100, 0.6), hsla(120, 100%, 50%, 0.5), ...

See more in CSS Color Module Level 4.

Absolute lengths

Type Value Assignment
px 1px = 1/96th of 1 inch
pt 1pt = 1/72nd of 1in
in 1in = 2.54cm = 96px

Extra tidbit: Learn about how pixels are defined across it as being physical or digital.

Relative-Length Values
(csswg spec)

  • Values computed relative to another value declared elsewhere.
  • Font-Relative: em & rem
  • Relative to Parent-Container Width Values: 75% (depends on rule being applied)
  • Relative to Viewport: vw & vh
  • Numerous others: ex, ch, etc.

Fixed vs. Absolute Positioning

  • Rough main difference:
    >> fixed positions an element based on the viewport.
    >> absolute positions an element based on the closest parent containing block.
